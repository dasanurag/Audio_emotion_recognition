Emotion recognition using speech data, complete with a GUI demo

Dependencies
============
- numpy
- playsound
- tensorflow
- keras
- matplotlib
- scipy
- seaborn
- librosa
- tkinter

sage
============
To install the necessary packages, run the following command:-<br/>
`pip install -r requirements.txt`

Run,
`python gui.py`
